# ng2-structured-seed

Angular2 seed with a directory and file structure designed to be discoverable if not always composable.
Directory Structure:

    + app
      + root
        + navbar 
          navbar.component.ts
        root.component.ts
      + shared
        + example
          example.component.ts
      main.ts (bootstrapping file)
    index.html
    package.json
    tsconfig.json
    typings.json
